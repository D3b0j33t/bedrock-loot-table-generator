## Running Locally

To test locally, you may use the flask server included in this project:

### First time setup
 - Create venv: `python -m venv venv`
 - `./venv/Scripts/Activate`
 - `pip install flask`

### Running
 - `set FLASK_APP=app`
 - `flask run`
 - Available at: `http://127.0.0.1:5000/`
